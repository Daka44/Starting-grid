# Starting Grid - F1 정보 제공 웹 서비스

F1 레이스 일정과 초보자를 위한 가이드를 제공하는 Django 웹 애플리케이션입니다.

## 주요 기능

### 1. F1 레이스 일정
- 전체 레이스 캘린더 제공
- 각 서킷별 상세 정보 (알면 재미있는 정보)

### 2. 초보자 가이드
- F1 관련 기초 지식
- 경기 규칙 설명
- 용어 해설

## 기술 스택

- Backend: Django 5.2
- Frontend: Bootstrap 5
- Database: SQLite3

## 개발 예정 기능

- [ ] 드라이버/팀 정보
- [ ] 실시간 순위 정보
- [ ] 레이스 날씨 정보 (날씨 API 연동)
- [ ] 사용자 커뮤니티
- [ ] F1 뉴스 피드
